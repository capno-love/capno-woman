
## Capno Woman Powers
Capno Woman derives her powers from her Prismatic Holder, an ancient artifact in the shape of a cigarette holder. When Jane Thomas lights up a cigarette that is inserted into Prismatic Holder, she transforms into Capno Woman and gains various powers.

### Powers

| Color | Power | Type | Behavior |
| --- | --- | --- | --- |
| Red | Tonfa | Construct | Capno Woman blows the smoke on her right wrist, forming a tonfa |
| Orange | Rotella | Construct | Capno Woman blows the smoke on her left wrist, forming a rotella |
| Yellow | Binding Tendrils | Cloud | Cloud can be redirected as a rope |
| Blue | Force Shield | Cloud | Force shield that stops projectiles and magic attacks |
| Purple | Disorienting Cloud | Cloud | Any who inhale the cloud become confused about their surroundings |
| Pink | Healing Redolence | Cloud | Heals wounds and health-draining effects |
| Light Blue | Freezing Mist | Cloud | Contact with an inanimate object will coat it in a sheet of ice |
| White | Dazzling Haze | Cloud | Acts like a smoke bomb |


#### Power Notes
 - All powers originate from an exhale and last for one minute before expiring
 - Capno Woman can only have one instance of each cloud active at a time 

#### Constructs
 - Constructs are indestructible
 - Only one construct can be active at a time as Capno Woman needs one hand for Prismatic Holder
    - Creating a second construct will deactivate the first

#### Cloud Powers
 - Capno Woman can move and reshape the cloud using her mind
 - Smaller clouds are more potent while larger clouds are weaker
 - Only one cloud of each color can be active at a time
    - Creating a second cloud of a color will deactivate the first
